Configure o ID da planilha e a API Key para começar.

🚀 Como configurar

1

Obtenha o ID da planilha

Abra sua planilha Google Sheets. Na URL, copie o trecho entre /d/ e /edit.
Ex: https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5n.../edit

2

Crie uma API Key gratuita

Acesse console.cloud.google.com, crie um projeto, ative a Google Sheets API e gere uma API Key.

3

Compartilhe a planilha

A planilha precisa estar pública para leitura (Compartilhar → Qualquer pessoa com o link pode ver), ou a API Key deve ter permissão de acesso.

4

Cole os dados acima e clique em Carregar

O sistema buscará automaticamente a aba ESCALA DE ENTREGA DD.MM do dia atual, ou da data selecionada.
